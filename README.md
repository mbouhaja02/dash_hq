# ShelfGuide - Dashboard HQ

Dashboard web autonome pour Vercel.

## Variables Vercel

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- aucune variable de filtre n'est requise pour la vue HQ

## Deploy Vercel

Dans Vercel, choisis ce dossier comme Root Directory: `dashboard-hq`.

Build command: `npm run build`

Output directory: `dist`
